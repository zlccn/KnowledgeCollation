# Git 常用命令

## 常用命令

`git add <file> | *` : 

把文件添加到暂存区，* 表示把所有修改都添加到暂存区

`git commit -m '...'` : 

把文件提交到仓库， '...' 中放置改动说明

`git status` : 

查看当前所有修改

`git diff` :

查看修改内容，输入后会打印出所有修改过的内容

`git log` : 

按照从前到后的顺序，查看提交版本；直接使用会打印出最完整的信息以及版本号，如果只想查找短号和说明内容，使用命令：`git log --pretty=oneline --abbrev-commit`

`git reflog` :

查看命令历史，以便查找制定版本的版本号；

git 的特点是只记录修改，所以在一般情况下，你所有的历史版本都将被保存，即使你回退过的也不例外，除非你执行过删除命令
